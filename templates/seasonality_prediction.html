Seasonality Prediction

Introduction

This tool helps you forecast future values of a time series based on its seasonal patterns. Is convenient for users who need to make data-driven decisions based on seasonal trends. It's especially useful for businesses wanting to forecast sales, inventory, or other key metrics that follow regular patterns.

Functionality

To use it, you need to upload an Excel file with just one column of numeric data and a header. You also need to choose a periodicity (e.g., 4 for quarterly data), which should be a divisor of the number of data points you have (excluding the header). The tool will then generate a table with the predictions of the next cycle and create three visualizations: a graph of the original time series, a comparison of the prediction and actual values for the last cycle, and a graph showing the original series along with the future predictions.
